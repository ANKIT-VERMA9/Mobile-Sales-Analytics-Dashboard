# 📱 Mobile Sales Dashboard | Power BI

## 📌 Project Overview

The **Mobile Sales Dashboard** is an interactive Power BI project designed to analyze mobile phone sales performance across different brands, products, regions, and customer segments. The dashboard provides business insights that help in understanding sales trends, customer behavior, and revenue generation.

---

## 🎯 Project Objective

The primary objective of this project is to transform raw sales data into meaningful business insights through data visualization and analytics, enabling data-driven decision making.

---

## 🛠️ Tech Stack Used

* **Power BI Desktop**
* **Power Query** for data cleaning and transformation
* **DAX (Data Analysis Expressions)** for calculated measures and KPIs
* **Data Modeling** using relationships between tables
* **CSV/Excel File Formats** for source data integration

---

## 📊 Data Source

The dataset used in this project was obtained from publicly available datasets provided by:

* **Kaggle**
* Other open-source dataset repositories and data-sharing platforms

---

## ✨ Features & Highlights

* Interactive dashboard with dynamic filters and slicers.
* Sales analysis by brand, product, and category.
* Revenue, quantity sold, and profit tracking through KPIs.
* Regional and customer-based sales performance analysis.
* Trend analysis using time-series visualizations.
* User-friendly and visually appealing dashboard design.

---

## 📈 Key Insights

* Identified top-performing mobile brands and products.
* Analyzed monthly and regional sales trends.
* Compared sales performance across different categories.
* Highlighted important KPIs to support business decisions.

---

## 🚀 Conclusion

This project demonstrates skills in **Power BI, Data Visualization, Data Modeling, DAX, and Business Intelligence reporting**, making it suitable for Data Analyst and Business Analyst portfolios.
